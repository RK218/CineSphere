# 🎬 CineSphere – Movie Recommendation Interface
Link:https://cinesphere-rkwu3wkys2rasibv8cpstu.streamlit.app/

CineSphere is a simple and interactive **movie recommendation interface** built using **Python and Streamlit**.
It allows users to discover movies based on their **preferred language and genre**, and instantly provides detailed information about the recommended films.

The application uses a curated movie dataset containing films from multiple industries including **Hindi, Telugu, Tamil, Kannada, and English cinema**.

Users can easily select a language and genre, receive a list of recommended movies, and then view additional details such as the **movie description, director, rating, and streaming platform**.

---

## 🚀 Features

* 🎥 **Movie Recommendation System**
  Suggests movies based on selected **language and genre**.

* 🌍 **Multi-Language Support**
  Includes movies from:

  * Hindi
  * Telugu
  * Tamil
  * Kannada
  * English

* 🎭 **Multiple Genres Available**

  * Action
  * Comedy
  * Sci-Fi
  * Horror
  * Fantasy

* 📖 **Detailed Movie Information**
  Each recommendation includes:

  * Movie description
  * Director
  * Rating
  * Streaming platform

* ⚡ **Interactive User Interface**
  Built with Streamlit to provide a clean and easy-to-use interface.

## ⚙️ Installation and Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/cine-sphere.git
cd cine-sphere
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Streamlit app

```bash
streamlit run app.py
```

The application will open in your browser.

## 🎯 How the Application Works

1. The user selects a **language**.
2. The user selects a **movie genre**.
3. The application filters the movie dataset.
4. Recommended movies are displayed.
5. The user can select a movie to view detailed information.
